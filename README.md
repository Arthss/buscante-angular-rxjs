# Buscante

Este projeto foi criado com [Angular CLI](https://github.com/angular/angular-cli) version 14.0.3. Baseado no projeto feito pela Alura no curso de Rxjs

## Development server

Execute `ng serve` para iniciar o servidor de desenvolvimento. Acesse `http://localhost:4200/`. Sempre que um arquivo for atualizado a aplicação será recarregada automaticamente.

## Further help
Para conhecer mais comandos e obter ajuda execute o Angular CLI usando `ng help` ou acesse a pagina online de comandos [Angular CLI](https://angular.io/cli) page.
